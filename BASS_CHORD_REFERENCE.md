# Bass String Notes Reference
## 4-String Bass (Standard Tuning: E-A-D-G)

### E String (Thickest/Lowest)
E(0), F(1), F#(2), G(3), G#(4), A(5), A#(6), B(7), C(8), C#(9), D(10), D#(11), E(12)

### A String
A(0), A#(1), B(2), C(3), C#(4), D(5), D#(6), E(7), F(8), F#(9), G(10), G#(11), A(12)

### D String
D(0), D#(1), E(2), F(3), F#(4), G(5), G#(6), A(7), A#(8), B(9), C(10), C#(11), D(12)

### G String (Thinnest/Highest)
G(0), G#(1), A(2), A#(3), B(4), C(5), C#(6), D(7), D#(8), E(9), F(10), F#(11), G(12)

## Common Shapes
- Major barre: [x, R, R+2, R+2] where R = root fret on A string
- Minor barre: [x, R, R+2, R+1] where R = root fret on A string
- 7th barre: [x, R, R, R] where R = root fret on A string
- Open triads use open strings where possible
